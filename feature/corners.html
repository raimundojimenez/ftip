Corner detection¶

Pour détecter les coins des objets dans une image, on peut commencer par déterminer les lieux où deux contours se joignent. Il existe toutefois d’autres méthodes, parmi lesquelles :

  • le détecteur de Moravec [Moravec 1980],

  • le détecteur de Harris [Harris 1988].
